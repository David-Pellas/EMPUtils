# EMPUtils

**EMPUtils** is a free, open-source Chrome extension that blocks ads, neutralizes trackers, and gives you full visibility into who is watching you browse — all processed locally on your device.

---

## Features

- **Ad Blocking** — Network-level, element-level, and JavaScript-level ad blocking using EasyList (30,000+ rules, refreshed every 24 hours)
- **Tracker Neutralization** — Intercepts and disables analytics and tracking scripts before they can run (Google Analytics, Facebook Pixel, Hotjar, and more)
- **Tracker Intelligence Database** — Built-in profiles for 228+ known trackers with risk levels, data collected, and ownership info
- **Domain Verification** — WHOIS/RDAP lookup, DNS infrastructure detection, and ownership record identification
- **Privacy Score** — Per-page 0-100 score with letter grade and tracker breakdown
- **Page Tracker Analysis** — View all external domains loaded on a page, filterable by category or risk level, exportable to CSV
- **Real-Time Alerts** — Dismissible banner when high-risk trackers are detected, with quick Whitelist or Block actions
- **Whitelist and Custom Block Controls** — Whitelist sites entirely or manually block specific domains
- **YouTube Ad Skipping** — Automatically skips YouTube ads
- **Customizable Theme** — Color picker to personalize the extension UI

---

## Installation

1. Download or clone this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** using the toggle in the top right.
4. Click **Load unpacked** and select the `EMPUtils` folder.
5. The extension icon will appear in your toolbar.
